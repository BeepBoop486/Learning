Welcome to the Learning documentation

Hello everybody! Welcome to the Learning documentation. This is made for people who want to contribute - or even create their own LMS platform based on the Learning source code.

This documentation is splitten in two parts:

  1. Creating and Managing courses
  2. Creating and Managing blog entries

Creating and Managing courses

Navigate to this section if you want to create your own courses. This section also contains instructions for modifying and updating already existing courses.

Creating and Managing blog entries

Go to this page if you are willing to create a blog entry. Here you can also find how to modify someone else's blog entry.
