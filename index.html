Features

Learn about some of the features we offer.

Use Learning Offline

Clone the source code of Learning and run it locally. With all its courses, all its images and styles.

Free support

Get support from instructors about a course, with programmers about the platform and any thing you need will and always be free.

Learn and teach

Do you find interesting the content of Learning? Enrol in a course. Do you know something which ain't available in Learning? You can create your own courses and make them available for other users to enrol.

Completely opensource

Read, modify and delete the source of Learning. You can even download it and create your own LMS platform. (As long as it stays GNU licensed)

Print your certification

After finishing every course, you can download a certification which could be easily printed.

Every course is created by the community

As the source of Learning is open, contributors can modify and create lessons and even entire courses and add them to the official Learning website
